# Professional-link-maker
Professional Link Maker

A simple Streamlit web app that helps clean and format long professional URLs.
It removes tracking parameters and improves readability for sharing links in
resumes, LinkedIn, emails, and portfolios.

Tech Stack:
- Python
- Streamlit

Features:
- Cleans long URLs
- Beginner-friendly implementation

How to Run:
pip install streamlit
streamlit run app.py
